# KsWare Presentation Framework

A MVVM Framework. 

or also >>*collected and coded knowledge about MVVM over the years*<< ;-)

organized in multiple parts:

- View-Framework
  - Converters 
  - Common 
- View-Model-Framework
- Business-Model-Framework

Use parts standalone or as big whole. 

Changes on demand.
